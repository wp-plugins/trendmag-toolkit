== Description ==

A specific plugin use in Trendmag Theme, included some custom widgets, and layout.

== Installation ==

1. Upload the files to the /wp-content/plugins/trendmag-toolkit/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance->Theme Options and use.
